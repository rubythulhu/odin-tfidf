# SuperLint Pro Max Plus Ultra

A linting plugin that judges your code silently.

## Installation

```bash
npm install superlint-pro-max-plus-ultra
```

## Configuration

Don't worry about it. SuperLint knows what you did.

## Features

- Finds errors you didn't know existed
- Creates new categories of errors just for you
- Passive aggressive warning messages
- Integration with everything (whether you want it or not)

## Usage

SuperLint will automatically run whenever you have hope in your heart. It will then systematically remove that hope.

## Common Warnings

- `WARN: This line is technically correct but I don't like it`
- `ERROR: You know what you did`
- `SUGGESTION: Have you considered a different career?`
