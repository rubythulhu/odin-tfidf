# Motivational Quotes for Developers

> "It works on my machine." - Ancient Proverb

> "Any code that is not tested is broken." - Murphy's Law, Developer Edition

> "The best error message is the one that never shows up." - Optimistic Developer, circa never

> "Real programmers don't comment their code. If it was hard to write, it should be hard to understand." - Someone who no longer works here

> "There are only two hard things in Computer Science: cache invalidation, naming things, and off-by-one errors." - Phil Karlton (Extended Edition)

> "It's not a bug, it's an undocumented feature." - Every developer ever

> "I would love to change the world, but they won't give me the source code." - Unknown

> "Programming is 10% writing code and 90% understanding why it doesn't work." - Sad but true

> "First, solve the problem. Then, write the code." - John Johnson (We skip straight to step 2)

> "Code never lies, comments sometimes do." - Ron Jeffries
